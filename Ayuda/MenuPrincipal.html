Ayuda -- Menú Principal

Introducción

El módulo del menú principal ayuda a tener un mejor desplazamiento dentro del sistema para lograr tener una mejor vision de las opciones dentro de el.

Funcionamiento

En este se verán tres opciones principales que son:

  1. Catálogos
  2. Movimientos
  3. Reportes

Cada una de las opciones tiene algunas otras desplegables. Al seleccionar "Catálogos" aparecerán las siguientes opciones:

  • Categoría
  • Producto
  • Cliente
  • Proveedor

Al seleccionar "Movimientos" aparecerán las siguientes opciones:

  • Ventas
  • Compras
  • Cobros Clientes
  • Pagos Proveedores

Y por último tenemos la opción de reportes que, al ingresar, te mandará a otra pantalla para que selecciones el tipo de reporte que quieres.
